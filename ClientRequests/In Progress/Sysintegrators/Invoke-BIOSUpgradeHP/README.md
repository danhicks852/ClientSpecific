# Overview
Updates UEFI HP Machine to latest firmware version

# Requirements
Windows 10+

# Process
Uses HP's firmware Library to update BIOS on the endpoint

# Payload Usage

```powershell
C:\>./Invoke-BIOSUpgradeHP.ps1
```
